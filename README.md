# Multi object tracking class

> Perform multi object tracking and use a custom function for tracked objects

MOT methods:

-   _detect_objects_ - detect object from a frame
-   _track_objects_ - perform a tracking update on frame

## Installation

```sh
pip install pymot
```

## Usage

```python
>>> from pycalc3.calculator import Calculator
>>> cal = Calculator()
>>> cal.add(12)
>>> cal.memory_value
12.0
>>> cal.subtract(3)
>>> cal.memory_value
9.0
>>> cal.multiply(3)
>>> cal.memory_value
27.0
>>> cal.divide(3)
>>> cal.memory_value
9.0
>>> cal.root(2)
>>> cal.memory_value
3.0
>>> cal.reset()
>>> cal.memory_value
0.0
```
